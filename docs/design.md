# Design System

## Colors
- Primary: #2E7D32 (agricultural green)
- Secondary: #FFB74D (warning)
- Error: #D32F2F (alert)
- Background: #F5F5F5

## Typography
- Font: Roboto (MUI default)
- Headings: 600 weight
- Body: 400 weight

## Components
- Device cards with status indicators
- Line charts for historical data
- Alert banners
